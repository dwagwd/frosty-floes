# frosty-floes

Static GitHub Pages site for the Frosty Floes brand and Frosty Floes: Sudoku public links.

## Files

- `index.html`: Frosty Floes landing page and brand/world overview
- `support.html`: dedicated support page suitable for App Store support URLs
- `privacy-policy.html`: Frosty Floes: Sudoku privacy policy
- `app-ads.txt`: ads.txt file for AdMob verification
- `assets/`: shared icon, CSS, and token files copied from the workspace `brand-system/` source

- Support URL: `https://dwagwd.github.io/frosty-floes/support.html`
- Privacy Policy URL: `https://dwagwd.github.io/frosty-floes/privacy-policy.html`
